# Indentation and formatting
- Indent with 4 spaces.
- Replace tabs with spaces, there are options for that in IDEs.
- Use LF (Unix) line endings.
```lua
for i = 1, 10 do
    for j = 1, 20, do
        print("Hello, world!")
    end
end
```
